Ejercicio 14b. Arrays - Cálculos con notas

Obtén información estadística a partir de las notas de los alumnos:

  • Nota media. Utilizar reduce() para obtener el promedio.
  • Nota mínima. Utilizar reduce() para obtener la nota mínima.
  • Nota máxima. Utilizar reduce() para obtener la nota máxima.
  • Número de aprobados. Utilizar filter() para obtener el número de aprobados.
  • Número de suspensos. Utilizar filter() para obtener el número de suspensos.
